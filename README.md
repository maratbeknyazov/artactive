# 🎨 ArtActive — Landing Page для «Угсуг»

Современный и адаптивный лендинг, разработанный для проекта **Угсуг**. Сайт представляет собой информационную страницу с акцентом на удобство регистрации пользователей через различные лид-формы.

**🔗 [https://maratbeknyazov.github.io/artactive/](https://maratbeknyazov.github.io/artactive/)**

-----

## 🚀 О проекте

Этот проект — результат работы над созданием легкого и отзывчивого интерфейса. Основная цель страницы — предоставить информацию и собрать заявки пользователей через интерактивные формы.

### Ключевые особенности:

  * **Множественные точки конверсии:** Несколько продуманных форм регистрации для разных сценариев.
  * **Полная адаптивность:** Сайт корректно отображается на смартфонах, планшетах и десктопах.
  * **Плавные анимации:** Использование JS и jQuery для улучшения пользовательского опыта.
  * **Оптимизированная верстка:** Быстрая загрузка благодаря чистому коду.

-----

## 🛠 Технологический стек

Проект построен с использованием классических и надежных инструментов фронтенд-разработки:

| Инструмент | Описание |
| :--- | :--- |
| **HTML5** | Семантическая структура страницы. |
| **SCSS** | Модульный и поддерживаемый CSS с использованием переменных и миксинов. |
| **JavaScript** | Логика взаимодействия и динамические элементы. |
| **jQuery** | Упрощенная манипуляция DOM и обработка событий форм. |

-----

## 📂 Структура проекта

  * `index.html` — основная разметка страницы.
  * `scss/` — исходные файлы стилей (архитектура стилей).
  * `css/` — скомпилированный и оптимизированный CSS.
  * `js/` — скрипты для валидации форм и анимаций.

-----

## ⚙️ Как запустить локально

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/maratbeknyazov/artactive.git
    ```
2.  Перейдите в папку проекта:
    ```bash
    cd artactive
    ```
3.  Откройте `index.html` в любом браузере или используйте расширение **Live Server** в VS Code.
